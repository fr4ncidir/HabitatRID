read_serial_packet reads a packet from serial port. The packet is identified with a starting byte. So, while reading the first time, we ignore all is coming before this packet and then we start read packet-by-packet. The packet has a fixed lenght.

Parameters
file_descriptoris the identifier for the serial port
fixed_lenghtis the exact number of bytes to be read
bufferis the place to store the bytes read
startis the starting byte of the packet
Returns
EXIT_FAILURE, EXIT_SUCCESS or NO_PACKET
